# 极简休息提醒器

25分钟工作 + 5分钟休息的番茄钟工具，跨平台支持 macOS / Windows / Linux。

## 特点

- 超低资源占用（仅用 tkinter，无其他依赖）
- 休息时全屏粉色界面 + 倒计时
- 休息结束等待鼠标点击或键盘按键才开始下一轮
- 支持重置 25 分钟工作时间

## 使用方法

### macOS / Linux
```bash
./run.sh
# 或
python3 rest_timer.py
```

### Windows
```cmd
run.bat
:: 或
python rest_timer.py
```

## 界面说明

- **绿色 "工作中"**：25分钟倒计时进行中
- **红色 "休息时间"**：5分钟全屏粉色休息界面
- **重置25分钟**：随时重新开始工作计时
- **立即休息**：跳过工作，立即进入休息

## 休息结束

休息时屏幕显示粉色背景和倒计时，时间到了后：
- 点击鼠标 或 按任意键 → 开始新的25分钟工作
